<p xmlns="http://www.w3.org/1999/xhtml">Earlier we created the contact form. Let’s spice it up a little bit — let’s create an automation for it! </p><p xmlns="http://www.w3.org/1999/xhtml">Since we might want to get an email notification when someone sends us a message through the contact form, let’s do that. Let’s create an automation that sends an email to us.</p><p xmlns="http://www.w3.org/1999/xhtml">We start by going into the contact form database and in the 3 dots menu you will see “Automations”.</p><img xmlns="http://www.w3.org/1999/xhtml" src="08239bbfe4029bb7_image.png" style="width: auto; --upload-progress: 0%;" id="af5aed3b-5e86-435a-a07f-d8a0aac7c0dd"><p xmlns="http://www.w3.org/1999/xhtml">And you will see an empty page of Automations. Let’s create one:</p><img xmlns="http://www.w3.org/1999/xhtml" src="04b2e0d8b5819e12_image.png" style="width: auto; --upload-progress: 0%;" id="e07afaab-6184-4f59-9447-42a1daf1fa85"><p xmlns="http://www.w3.org/1999/xhtml">Next you will see a few things, which I will explain here:</p><img xmlns="http://www.w3.org/1999/xhtml" src="ff005d251444bf6d_image.png" style="width: auto; --upload-progress: 0%;" id="00977def-cb73-4142-a35b-7692167fa8ff"><ol xmlns="http://www.w3.org/1999/xhtml"><li><p>The Automation will be triggered by some event. At the moment, only <code>Row inserted</code> is supported.</p></li><li><p>Triggering table. We need to specify, which table triggers the Automation.</p></li><li><p>Bindings. When the row is inserted, it has some data, and the different fields are named by some human-readable name which tells what the data is about. Here are those names turned into lowercase character strings, wrapped in <code>{{}}</code>. You can use these bindings in the email that you send out. For example, the <code>{{name}}</code> means the name that the person gave and <code>{{email}}</code> is his/her email address.</p></li><li><p>Publishing. When you are ready to take this Automation into use, set this to Public.</p></li><li><p>Add action. Your automation needs actions — at least one. Add it here.</p></li></ol><h2 xmlns="http://www.w3.org/1999/xhtml">Actions</h2><p xmlns="http://www.w3.org/1999/xhtml">Your automation needs actions. Currently, two types of actions are supported: </p><ol xmlns="http://www.w3.org/1999/xhtml"><li><p>Sending an email</p></li><li><p>Creating a task</p></li></ol><img xmlns="http://www.w3.org/1999/xhtml" src="3b4d8f00a0f58d5f_image.png" style="width: auto; --upload-progress: 0%;" id="a4165404-d25c-4644-a22f-3c6ce650af52"><p xmlns="http://www.w3.org/1999/xhtml">And email needs a sender. And we can only use a sender that we have specified by ourselves. So, in case you haven’t set up Emails yet, now would be a good time to do that.</p><p xmlns="http://www.w3.org/1999/xhtml">Here are the email accounts that I have set up, and I can choose from one of them:</p><img xmlns="http://www.w3.org/1999/xhtml" src="0d0877d392999828_image.png" style="width: auto; --upload-progress: 0%;" id="3566e9a6-7acd-47e6-a39f-8fe7376ccab0"><p xmlns="http://www.w3.org/1999/xhtml">Next we get to the main point — writing the email template that will be used for sending.</p><img xmlns="http://www.w3.org/1999/xhtml" src="a9dd3996432b0a05_image.png" style="width: auto; --upload-progress: 0%;" id="7d779717-651d-4a53-9aa4-397877c13371"><p xmlns="http://www.w3.org/1999/xhtml">Here is the screen for writing email templates:</p><img xmlns="http://www.w3.org/1999/xhtml" src="130586afb140b073_image.png" style="width: auto; --upload-progress: 0%;" id="a408b2c6-85ac-48ec-8156-10bd5a581947"><p xmlns="http://www.w3.org/1999/xhtml">The <em>Bindings</em> are explained there again — how they work. You can use them in the body section of the email. And the actual bindings you saw in previous section.</p><p xmlns="http://www.w3.org/1999/xhtml">The <em>To</em> field would be the email where you want the email to arrive at. The <em>From</em> field was set earlier.</p><p xmlns="http://www.w3.org/1999/xhtml">You can write a template like this, for example:</p><p xmlns="http://www.w3.org/1999/xhtml">Subject: Contact form submission<br>To: youraddress@gmail.com<br>Body:</p><pre xmlns="http://www.w3.org/1999/xhtml"><code class="language-html">Hi me!

Someone just filled our contact form with this information:

Name: {{name}}
Email: {{email}}
Message: {{message}}</code></pre><p xmlns="http://www.w3.org/1999/xhtml">Click <em>Save template</em>.</p><p xmlns="http://www.w3.org/1999/xhtml">At this point just change the <em>Publishing</em> to <em>Public</em>. When you do that, you can test the Automation:</p><img xmlns="http://www.w3.org/1999/xhtml" src="457cad1ffa5d5875_image.png" style="width: auto; --upload-progress: 0%;" id="4542b29a-3582-4230-9c29-feebc4d12cb1"><p xmlns="http://www.w3.org/1999/xhtml">Then just start waiting for the emails to pour in!</p><p xmlns="http://www.w3.org/1999/xhtml"></p>
