<p xmlns="http://www.w3.org/1999/xhtml">Let’s see how we can use Aamu.app’s Form Builder to create survey forms. </p><p xmlns="http://www.w3.org/1999/xhtml">First we go to Forms section and create an empty form:</p><img xmlns="http://www.w3.org/1999/xhtml" src="5365f206da75c395_image.png" style="width: auto; --upload-progress: 0%;" id="b96036d7-5971-4959-9e86-40c56ae229a0"><p xmlns="http://www.w3.org/1999/xhtml">Which looks like this:</p><img xmlns="http://www.w3.org/1999/xhtml" src="bde2653c7845aa21_image.png" style="width: auto; --upload-progress: 0%;" id="f3c3b3a5-a7fe-4333-8e83-95a3ea8c27fa"><p xmlns="http://www.w3.org/1999/xhtml">Start by adding the Form title and Description, if you so choose.</p><p xmlns="http://www.w3.org/1999/xhtml">After which you can start adding form fields.</p><img xmlns="http://www.w3.org/1999/xhtml" src="4f36383b0e04528c_image.png" style="width: auto; --upload-progress: 0%;" id="c3135cc4-4fe7-48a7-b726-9f19eae2b470"><p xmlns="http://www.w3.org/1999/xhtml">Text answer and Choice is probably the most common. Let’s add a Choice:</p><img xmlns="http://www.w3.org/1999/xhtml" src="38306a2372dcfb90_image.png" style="width: auto; --upload-progress: 0%;" id="6654df55-1427-4f6b-af40-db7ca4a13893"><p xmlns="http://www.w3.org/1999/xhtml">With Choice, you have three different ones: radio buttons (select one from many), dropdown box (select one from many) and checkboxes (select many from many). Here we have radio buttons, and it looks like this as the final form:</p><img xmlns="http://www.w3.org/1999/xhtml" src="b6f9f0a08449851c_image.png" style="width: auto; --upload-progress: 0%;" id="3dd492bc-0fc5-4fb7-9707-b2b1af86f931"><p xmlns="http://www.w3.org/1999/xhtml">Let’s add some data (fill the form and Submit it), then look at the Responses page:</p><img xmlns="http://www.w3.org/1999/xhtml" src="3b26b43ffad64084_image.png" style="width: auto; --upload-progress: 0%;" id="5a729ef0-7b61-457f-918c-6c421c097fbc"><p xmlns="http://www.w3.org/1999/xhtml">You get a nice graphic of the answers. You can see the detailed data in Databases:</p><img xmlns="http://www.w3.org/1999/xhtml" src="d17534efa4cd0bff_image.png" style="width: auto; --upload-progress: 0%;" id="357d3a9a-599f-48d8-bc36-65ab7acc7bef"><p xmlns="http://www.w3.org/1999/xhtml">In Form settings you can set all kinds of useful behavior, like a friendly URL, a custom thank-you page or custom CSS:</p><img xmlns="http://www.w3.org/1999/xhtml" src="9b78985f5e9632ce_image.png" style="width: auto; --upload-progress: 0%;" id="b7abac3a-9ac9-44ee-9217-1a2c07dec3dc"><h2 xmlns="http://www.w3.org/1999/xhtml">Forms and Databases</h2><p xmlns="http://www.w3.org/1999/xhtml">Since each form submission creates a database entry, all the features, that are available for databases, are also available for form data. One particularly useful feature may be Database automations — which means that when a new database row (form submission) is added, you can define different actions for it. One action may be to send an email, another may be to create a Task.</p><p xmlns="http://www.w3.org/1999/xhtml">We will look at Database automations next!</p><p xmlns="http://www.w3.org/1999/xhtml"></p>
