<p xmlns="http://www.w3.org/1999/xhtml">Everywhere that you write <em>content</em> in Aamu.app, i.e. comments, tasks, docs etc., can can add <em>internal links</em> with the slash menu:</p><img xmlns="http://www.w3.org/1999/xhtml" src="561584338b04cad8_image.png" style="width: auto; --upload-progress: 0%;" id="29421ff1-900e-41e9-a6f0-d7f048b315ec"><p xmlns="http://www.w3.org/1999/xhtml">There you are presented with a dialog window, which has a search input box:</p><img xmlns="http://www.w3.org/1999/xhtml" src="43c65776b4784f79_image.png" style="width: auto; --upload-progress: 0%;" id="f6d27260-47a0-4bc6-867e-0cabbb0866ed"><p xmlns="http://www.w3.org/1999/xhtml">Here my search term is “search for” and it is currently searching for tasks. I can search for any item types in Aamu.app. Switching between tabs can be done with a mouse (obviously), but also with the shortcut <code>Shift + left</code> and <code>Shift + right</code>.</p><p xmlns="http://www.w3.org/1999/xhtml">When you click the item that you want to create a link to, this will be added to the document:</p><img xmlns="http://www.w3.org/1999/xhtml" src="265914a44ba59c0e_image.png" style="width: auto; --upload-progress: 0%;" id="400092b6-6b18-4f5c-b818-e49d5aa616d2"><p xmlns="http://www.w3.org/1999/xhtml">It will add a familiar box into the document/comment that you are currently writing. </p><p xmlns="http://www.w3.org/1999/xhtml">Clicking the box will take you to that item. Pressing <code>Esc</code> will take you back.</p><p xmlns="http://www.w3.org/1999/xhtml">This is how easy it is to reference other items in Aamu.app. You can link from everywhere to anywhere!</p><p xmlns="http://www.w3.org/1999/xhtml"></p>
